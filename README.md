Capital Pulse - Tasks 1 & 2
Description
This is the compulsory part of the GDG AI/ML project: A stock analysis system.

Task 1: LSTM forecasting with dashboard for predictions and metrics.
Task 2: Chatbot for stock queries using news and data.

How to Run
Clone repo: git clone https://github.com/YOUR_USERNAME/CapitalPulse-AI-Stock-Intelligence.git
Install: pip install -r requirements.txt
Task 1: cd task1_predictive_core && streamlit run predictive_core.py
Task 2: cd task2_analytical_chatbot && streamlit run chatbot.py (add NewsAPI key in tools.py)
